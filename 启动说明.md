### 🔧 手动启动方式（推荐）

#### 1. 后端服务（已在启动）
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload
```

#### 2. 前端服务（待后端启动完成后执行）
```bash
cd frontend
npm install
npm run dev
```

### 📞 访问地址
- **后端API**: http://localhost:8000
- **前端应用**: http://localhost:5173
- **API文档**: http://localhost:8000/docs

### 🎯 功能测试
1. **简历优化**: 输入简历文本，获取 AI 优化建议
2. **模拟面试**: 与 AI 面试官实时对话
3. **八股题库**: 练习面试题目，获取 AI 评分

### ⚠️ 注意事项
- 首次启动需要安装依赖，请耐心等待
- 后端使用 SQLite 数据库，会自动创建
- 默认使用模拟 AI 响应，如需真实 OpenAI API，请配置环境变量
- 支持 Chrome、Firefox、Safari 等现代浏览器

### 🆘 问题排查
- 端口被占用：修改 `vite.config.ts` 或启动命令中的端口
- 依赖安装失败：清除缓存后重试
- AI 功能异常：检查网络连接或 API Key 配置
